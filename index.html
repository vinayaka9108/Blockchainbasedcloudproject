for Cloud Data Storage

A decentralized, tamper-proof cloud storage security system combining AES-256 encryption, blockchain immutability, and smart contract access control to protect sensitive data at every step.

System Workflow

How your data travels securely from upload to verification

1. File Upload

User selects and uploads a file through the secure web interface

2. AES-256 Encryption

File is encrypted with AES-256-GCM. The key is wrapped with RSA-2048

3. Cloud Storage

Only the encrypted file is stored. Plain text never touches the server disk

4. SHA-256 Hashing

A unique cryptographic hash of the original file is computed for fingerprinting

5. Blockchain Record

A new block containing the file hash and metadata is added to the immutable chain

6. Smart Contract

Access control rules are enforced automatically before any file operation

7. Integrity Check

On download, SHA-256 hash is recomputed and compared against the blockchain record
